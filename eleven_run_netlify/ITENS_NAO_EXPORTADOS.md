# Itens não exportados

## Arquivos do projeto

**Nenhum arquivo rastreado da versão 15 ficou de fora.** Todo o código-fonte, imagens, sprites, personagens, cenários, obstáculos, fontes, configurações, scripts, testes e declarações de dependências disponíveis no projeto foram incluídos.

## Diretórios operacionais não incluídos

Os itens abaixo não fazem parte do código ou dos assets do jogo e são recriados automaticamente:

- `.git/`: metadados internos e histórico do repositório usado pela hospedagem. O commit de origem foi registrado no `README.md`.
- `node_modules/`: cópias instaladas das dependências. Todas as dependências e suas versões exatas estão preservadas em `package.json` e `package-lock.json` e podem ser restauradas com `npm install`.
- `.sites-runtime/`: cache temporário da plataforma de hospedagem, específico do ambiente de trabalho.

## Áudio

Não havia arquivos externos de som ou música para exportar. A trilha e todos os efeitos são sintetizados em tempo real pelo Web Audio API no arquivo `app/page.tsx`, que está incluído no backup.
